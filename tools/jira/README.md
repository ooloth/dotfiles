Install via homebrew:
https://github.com/ankitpokhrel/jira-cli/wiki/Installation#homebrew

Get Jira API token:
https://id.atlassian.com/manage-profile/security/api-tokens

Save it in 1Password
Export JIRA_API_TOKEN in shell

Run `jira init`
